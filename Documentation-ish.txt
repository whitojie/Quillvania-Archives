Git Stuff
1. Initialize repo, git init
2. add files to staging, git add .
3. Make commit, git commit -m "Initial commit"
4. Create main branch, git branch -M main
5. Add Github remote, git remote add origin https://github.com/whitojie/Quillvania-Archives.git (make sure to make repo in github)
6. Push to github, git push -u origin main
